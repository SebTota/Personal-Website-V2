reCAPTCHA Solver

Google, "I'm not a robot", reCAPTCHA solver built on top of an open-source project called Buster. Once a website is specified, it will complete the reCAPTCHA using Wit.ai's speech-to-text API. Finally, it will store the solution in a database for client retrieval. Each unique solution is valid for three minutes. Assuming the target website on which a reCAPTCHA bypass is needed is known in advance, the client can request workers in advance, then receive the solution when needed allowing an almost instantaneous bypass.

  • Python
