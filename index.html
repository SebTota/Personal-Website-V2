Running Form Analysis

TrueForm

TrueForm is a Python application used to estimate running form from 2D video using Human Pose Estimation. It uses OpenPose, a real-time multi-person system to detect human body keypoints on images to estimate the major joints in the body, from which it is then able to analyse a runners form. The application aims to improve the performance of long-distance runners, while decreasing the risk of injury with easily accessible technology.

Measurements include: cadence, stride length, leg and arm swing symmetry, oscillation, angle of knee upon impact, and foot strike among other measurements. All these measurements are then analysed to give suggestions on improving running form.

Temple University

Class Scheduling Tool

Official Website

A full-stack application focused on simplifying the course scheduling process for Temple University students.

After specifying desired courses, students can view all the possible schedules for a given semester without time conflicts. Students can also enter desired times they are not available during the week to avoid scheduling conflicts with events outside of school.

One of the major deciding factors for students choosing which section of a class to take depends on the professor's past performance. With a verified Temple University student review system in place, students with a Temple University email address can leave an anonymous professor review to aid the decision process for future students.

noReCap

reCAPTCHA Solver

The noReCap reCaptcha solver takes the frustration our of Google, "I'm not a robot", reCAPTCHA by allowing users to bypass it completely. The solver is built on top of an open-source project called Buster.

Once a website is specified, noReCap automates the reCaptcha solving process using Wit.ai's speech-to-text API in just seconds. By storing the solution in a database, the end-user can easily integrate noReCap into any automation script using the easy to use API.

If a website domain for which a reCaptcha solution is needed is known in advance, the client can request solutions in advance allowing almost instantaneous bypass of Google's reCaptcha.
